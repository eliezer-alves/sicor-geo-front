# SICOR Geo Frontend\n\nFrontend moderno para visualizacao geoespacial de operacoes de credito rural (SICOR/BCB).\n\n## Stack\n\n- React 19 + TypeScript + Vite\n- TailwindCSS v4\n- Leaflet (react-leaflet)\n- React Query (TanStack)\n- React Router\n\n## Setup\n\n```bash\nnpm install\nnpm run dev\n```\n\nO frontend roda em `http://localhost:5173` e faz proxy de `/api` para `http://localhost:8000` (backend Django).\n\n## Build\n\n```bash\nnpm run build\n```
